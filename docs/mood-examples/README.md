# Mood Example Corpus

This folder contains the expanded research corpus for each mood.

Each mood file includes:

- a reflective opening paragraph
- a long-form narrative story
- 25 life situations
- 25 human nature examples
- 25 historical event examples
- 25 natural occurrence examples
