# Energetic

## What this mood is
Energetic is a high-activation state marked by momentum, readiness, and drive. It often feels like having extra fuel for action.

## How it often feels
- Alert and activated
- Motivated to start or finish tasks
- Physically and mentally "switched on"

## Human function
Energetic mood supports initiative, persistence, and problem-solving under pressure.

## Watch-outs
If unregulated, it can tip into impatience, overcommitment, or scattered focus.

## Musical Build Guide (Ableton Live, concept-only)

### Harmonic palette
- Tempo target: `110-138 BPM`
- Scale and mode options: `Dorian`, `Mixolydian`, `Aeolian`
- Chord colors: power-chord plus color (`add9`), `sus` dominants, short `m9` hits
- Simple voicing concepts:
  - Bass: octave roots or root-fifth drive
  - Harmony: tight triads and `sus` voicings in the mid register

### Starter progression concepts
- `i - bVII - bVI - bVII`
- `I5(add9) - bVII - IV/3 - I`

### Cross-cultural concept cues (principle transfer)
- Build intensity through repetition plus escalation.
- Use call-and-response between parts.
- Add density in layers every 8 or 16 bars, not all at once.
