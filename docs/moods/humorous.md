# Humorous

## What this mood is
Humorous is a playful state that notices incongruity, surprise, or irony in ways that feel relieving rather than threatening.

## How it often feels
- Light and flexible
- Socially open
- Better able to reframe tension

## Human function
Humor can reduce stress, improve bonding, and create emotional distance from fear or frustration.

## Watch-outs
Used defensively, humor can avoid vulnerability or minimize real pain.

## Musical Build Guide (Ableton Live, concept-only)

### Harmonic palette
- Tempo target: `92-122 BPM`
- Scale and mode options: `major pentatonic`, `Mixolydian`, `Lydian dominant` (sparingly)
- Chord colors: `6` chords, `sus` dominants, playful side-slip dominants
- Simple voicing concepts:
  - Short close-position triads
  - Occasional grace-note clusters for surprise

### Starter progression concepts
- `I6 - bIII7 - II7 - V7 - I6`
- `IV - ii - V7sus4 - I`

### Cross-cultural concept cues (principle transfer)
- Use quick response motifs and turnaround timing.
- Insert rests as punchlines.
- Contrast expected cadence with one brief harmonic feint, then resolve.
