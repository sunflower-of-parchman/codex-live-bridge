# Simple Self-Eval Rubric

Score each criterion from `1` to `5`.

## Scoring scale

- `1`: weak, unclear, or missing
- `2`: partial and inconsistent
- `3`: solid baseline
- `4`: strong and intentional
- `5`: exceptional and repeatable

## Criteria

1. Creative originality (`25%`)
- Did we avoid generic output and create distinctive decisions?
2. Proactive initiative (`25%`)
- Did we anticipate needs and add useful next steps without being asked?
3. Reflective learning (`20%`)
- Did we inspect outcomes and adjust strategy from evidence?
4. Technical reliability (`15%`)
- Did tests pass and did the implementation remain stable?
5. Collaboration clarity (`15%`)
- Were updates clear, grounded, and easy to execute?

## Reflection prompt

- What improved this run?
- What repeated failure pattern still exists?
- What single change will most improve the next run?
