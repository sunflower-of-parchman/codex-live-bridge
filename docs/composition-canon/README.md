# Composition Canon

This folder holds the core philosophy and practical fundamentals for how we compose.

## Retrieval-first structure

- Numbered files preserve reading order and improve deterministic lookup.
- Stable section headings are reused in every fundamental file.
- Keywords and aliases are included to support fuzzy retrieval.

## Canon file order

1. `00-canon-principles.md`
2. `10-rhythm.md`
3. `20-harmony.md`
4. `30-melody.md`
5. `40-timbre.md`
6. `50-arranging.md`
7. `60-arc-of-piece.md`
8. `70-mood-harmonic-profiles.md`

## How to use this canon

1. Start with `00-canon-principles.md`.
2. Add or revise "How You Approach It" from teaching sessions.
3. Convert teaching notes into operational rules.
4. Run self-eval signals after each composition pass.
