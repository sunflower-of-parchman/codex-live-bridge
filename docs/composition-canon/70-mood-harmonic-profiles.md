# Mood Harmonic Profiles

## Principles

- Mood drives harmonic behavior before chord decoration.
- Profiles are concept-only and key-agnostic.
- Functional movement, voicing shape, and density create emotional identity together.
- Cross-cultural influence is applied as principle transfer, not melodic borrowing.

## How You Approach It

Document your mood-harmony method here:

- choose one target mood
- choose one mode family and one functional movement pattern
- define voicing density and register plan
- define the tension/release arc over sections

## Operational Rules

1. Keep all guidance key-agnostic (`Roman numerals`, mode families, function roles).
2. Start with one primary profile and at most one secondary blend profile.
3. Set tempo frame and harmonic density before writing detailed notes.
4. Use profile-appropriate chord colors (`maj9`, `m9`, `sus`, altered dominants) intentionally.
5. If using cross-cultural context, transfer structural principles (arc, pacing, repetition, space), not direct phrase copying.

## Teaching Notes

### Ambient

- Tempo frame: `55-78 BPM`
- Mode families: `Dorian`, `Lydian`, `minor pentatonic`
- Functional movement: `i -> IV -> i -> bVII` or tonic pedal with upper-planing shifts
- Color palette: `m9`, `maj7(add9)`, `sus2`, quartal harmony
- Voicing concept: tonic-fifth bass drone with sparse upper `3/7/9` color
- Arc cue: very slow harmonic rate; long sustain
- Principle transfer: space-first pacing and subtle cyclic variation

### Beautiful

- Tempo frame: `68-92 BPM`
- Mode families: `Ionian`, `Lydian`, melodic-minor color borrowing
- Functional movement: `I -> vi -> IV -> V` with softened cadences
- Color palette: `maj9`, `6/9`, `m11`, gentle `sus4 -> 3` resolutions
- Voicing concept: open bass, lyrical upper voice-leading, controlled inner movement
- Arc cue: calm entry, bloom, gentle return
- Principle transfer: restraint, tone quality, and long-form contour

### Energetic

- Tempo frame: `110-138 BPM`
- Mode families: `Dorian`, `Mixolydian`, `Aeolian`
- Functional movement: `i -> bVII -> bVI -> bVII` or dominant loops
- Color palette: `add9` on power structures, short `sus` dominant hits
- Voicing concept: strong octave/root-fifth bass with compact mid-register stacks
- Arc cue: add harmonic density in layers every 8-16 bars
- Principle transfer: repetition plus escalation and call-response

### Hopeful

- Tempo frame: `82-108 BPM`
- Mode families: `Lydian`, `Ionian`, `Dorian`
- Functional movement: `I -> V/3 -> vi -> IV` or uplift cycles with suspended cadences
- Color palette: `maj9`, `add6`, `sus2`, light secondary-dominant pull
- Voicing concept: upward top-line tendency and open thirds/ninths
- Arc cue: unresolved optimism before final grounding
- Principle transfer: gradual emotional rise with collective refrain logic

### Humorous

- Tempo frame: `92-122 BPM`
- Mode families: major pentatonic, `Mixolydian`, selective dominant color shifts
- Functional movement: playful sidestep dominants returning to clear tonic
- Color palette: `6` chords, `sus` dominants, brief chromatic side-slips
- Voicing concept: compact triads with quick register flips and rests
- Arc cue: surprise-release cycles with clear returns
- Principle transfer: timing, turnaround wit, and strategic silence

### Intense

- Tempo frame: `120-160 BPM` (or `60-72 BPM` halftime)
- Mode families: `Phrygian`, harmonic-minor family, `Phrygian dominant`
- Functional movement: `i -> bII -> i -> V7(b9)` and diminished connectors
- Color palette: altered dominants, clustered dissonance, tritone pressure
- Voicing concept: low bass anchors plus upper minor-second/tritone friction
- Arc cue: prolonged tension before hard or partial release
- Principle transfer: ritual-like escalation and threshold pacing

### Mysterious

- Tempo frame: `65-95 BPM`
- Mode families: `Dorian b2`, `Phrygian dominant`, diminished fragments
- Functional movement: suspended tonic areas with ambiguous shifts
- Color palette: `sus2`, `sus4`, `maj7(#11)`, no-third structures
- Voicing concept: root-fifth bass ambiguity plus stacked fourths
- Arc cue: delayed clarification and selective resolution
- Principle transfer: ambiguity tolerance, sparse ornament, silence as structure

### Nostalgia

- Tempo frame: `70-96 BPM`
- Mode families: major with borrowed minor subdominant, natural-minor blend, `Mixolydian b6` color
- Functional movement: `I -> V/3 -> vi -> ivm -> I/5 -> V`
- Color palette: `maj7`, `m7`, borrowed `ivm6`, passing diminished links
- Voicing concept: warm mid-register voicings with inner semitone drift
- Arc cue: warmth-plus-ache, partial cadence, remembered return
- Principle transfer: bittersweet blend and motif recurrence with variation

### Rhythmic

- Tempo frame: `96-124 BPM`
- Mode families: `Dorian`, `Aeolian`, minor pentatonic
- Functional movement: long vamps (`i7 <-> IV7`) and low-churn harmonic loops
- Color palette: shells, suspended extensions, static harmony with rhythmic evolution
- Voicing concept: shell bass roles and interlocking upper dyads
- Arc cue: harmonic stability while rhythmic complexity grows
- Principle transfer: cycle design, interlocking parts, polymetric overlays

### Sad

- Tempo frame: `58-82 BPM`
- Mode families: `Aeolian`, `Dorian`, `Phrygian`
- Functional movement: `i -> bVI -> iv -> Vsus4 -> V` and minor plagal motion
- Color palette: `iiø7`, `m9`, suspended dominants, soft unresolved extensions
- Voicing concept: wide spacing and persistent upper tension tone (`9` or `11`)
- Arc cue: slow descent, delayed cadence, low dynamic ceiling
- Principle transfer: lament contour, breath-like pacing, gentle taper endings

### Live Implementation Checklists (5 steps each)

#### Ambient

1. Set project tempo in `55-78 BPM` and keep meter stable.
2. Choose one mode family (`Dorian`, `Lydian`, or minor pentatonic) and lock it for the first draft.
3. Build a low-churn loop from `i -> IV -> i -> bVII` or tonic-pedal plus upper-planing movement.
4. Voice with tonic-fifth bass drones and sparse upper `3/7/9` color tones.
5. Automate long, slow changes (filter, send, width) and confirm harmonic rate stays minimal.

#### Beautiful

1. Set tempo in `68-92 BPM` with moderate pulse and gentle subdivision.
2. Choose `Ionian` or `Lydian` as the base mode family and define one borrowed color rule.
3. Sketch `I -> vi -> IV -> V` movement with softened cadence treatment.
4. Use open lower voicings and smooth upper voice-leading centered on `3-7-9` color.
5. Arrange arc as calm entry, bloom, and gentle return; verify no section feels over-dense.

#### Energetic

1. Set tempo in `110-138 BPM` and pick a clear rhythmic anchor early.
2. Choose `Dorian`, `Mixolydian`, or `Aeolian` and keep harmonic language consistent.
3. Program a driving loop (`i -> bVII -> bVI -> bVII` or dominant-cycle variant).
4. Stack compact mid-register voicings over strong octave or root-fifth bass.
5. Increase harmonic or textural density every 8-16 bars to create escalation.

#### Hopeful

1. Set tempo in `82-108 BPM` and define a stable groove with light lift.
2. Choose a bright-leaning mode family (`Lydian` or `Ionian`, optionally `Dorian` blend).
3. Use uplift movement (`I -> V/3 -> vi -> IV`) with suspended or softened cadences.
4. Keep top voice rising stepwise where possible; emphasize open `3` and `9` colors.
5. End sections with partial resolution so forward momentum remains active.

#### Humorous

1. Set tempo in `92-122 BPM` with room for rhythmic turnarounds.
2. Choose major pentatonic or `Mixolydian` base and reserve one surprise-color move.
3. Write a short progression with one deliberate sidestep dominant and clear return.
4. Use compact voicings, rests, and quick register shifts to create timing contrast.
5. Validate that surprise events resolve quickly and read as playful, not chaotic.

#### Intense

1. Choose either fast (`120-160 BPM`) or halftime-heavy (`60-72 BPM`) tempo framing.
2. Select a high-tension mode family (`Phrygian`, harmonic-minor family, or `Phrygian dominant`).
3. Build around `i -> bII -> i -> V7(b9)` plus controlled diminished connectors.
4. Layer low anchors with upper minor-second or tritone friction before release.
5. Automate long tension ramps and delay final cadence until late in the section arc.

#### Mysterious

1. Set tempo in `65-95 BPM` and avoid overly busy rhythmic grids.
2. Choose `Dorian b2`, `Phrygian dominant`, or diminished-fragment language.
3. Use suspended tonic zones with ambiguous shifts instead of hard functional closure.
4. Keep bass simple (root-fifth) and stack fourth-based upper voicings.
5. Resolve selectively; verify ambiguity persists through most of the phrase.

#### Nostalgia

1. Set tempo in `70-96 BPM` with relaxed pulse and humanized timing.
2. Choose major/minor blend strategy with borrowed minor-subdominant color.
3. Shape movement like `I -> V/3 -> vi -> ivm -> I/5 -> V` with partial cadences.
4. Use warm mid-register voicings and gentle inner semitone drift.
5. Repeat one recognizable motif with small variation to imply remembered return.

#### Rhythmic

1. Set tempo in `96-124 BPM` and define primary cycle length first.
2. Choose `Dorian`, `Aeolian`, or minor pentatonic and keep harmony low-churn.
3. Start with long vamp behavior (`i7 <-> IV7`) before adding extra harmonic events.
4. Distribute interlocking rhythmic parts around shell-style harmonic support.
5. Add polymetric accent overlays (`3:2` or `4:3`) while preserving harmonic clarity.

#### Sad

1. Set tempo in `58-82 BPM` and keep dynamic floor low.
2. Choose `Aeolian`, `Dorian`, or `Phrygian` and define one descending contour rule.
3. Write movement around `i -> bVI -> iv -> Vsus4 -> V` with delayed cadence timing.
4. Space voicings widely and carry one unresolved extension (`9` or `11`) through changes.
5. Taper endings gently and confirm the arc feels reflective rather than merely slow.

## Self-Eval Signals

- Harmonic profile matches intended mood without relying on key-specific tricks.
- Voice-leading and density choices reinforce the selected profile.
- Tension/release arc is intentional at section level, not accidental bar-to-bar.
- Cross-cultural influences remain principle-based and respectful.
- Listener can identify mood identity within the first structural phrase.
