# Canon Retrieval Map

Use this map when routing prompts or lookups to the right canon file.

## Query routing

- groove, pulse, meter, syncopation -> `10-rhythm.md`
- chords, progression, voicing, tension/release -> `20-harmony.md`
- motif, contour, phrase, melodic range -> `30-melody.md`
- texture, color, patch choice, spectral clarity -> `40-timbre.md`
- layering, role assignment, transitions -> `50-arranging.md`
- form, narrative pacing, climax, ending -> `60-arc-of-piece.md`
- mood harmony, mood profile, chord mood mapping, emotional harmony -> `70-mood-harmonic-profiles.md`

## Retrieval conventions

1. Prefer exact file sections: `Principles`, `How You Approach It`, `Operational Rules`, `Teaching Notes`, `Self-Eval Signals`.
2. If a query spans multiple domains, start with `00-canon-principles.md`, then join two fundamentals.
3. Preserve canonical terms and aliases (`hamony`, `arch`) for fuzzy-matching compatibility.
