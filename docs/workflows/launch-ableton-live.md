# Launch Ableton Live

Use the `launch-ableton-live` skill on macOS to open Ableton with one command.

## Skill source

- `/Users/michaelwall/.codex/skills/launch-ableton-live/SKILL.md`

## Default launch command

```bash
/Users/michaelwall/.codex/skills/launch-ableton-live/scripts/launch-ableton-live.sh
```

## Force a specific app name

```bash
ABLETON_APP_NAME="Ableton Live 12 Suite" /Users/michaelwall/.codex/skills/launch-ableton-live/scripts/launch-ableton-live.sh
```

## Verification

- A successful run prints: `Launching Ableton Live: <app name>`
